WSO2 Enterprise Integrator - Message Broker Profile

WSO2 MB is a lightweight and easy-to-use Open Source Message Broker Server (MB) available under the Apache Software License v2.0.

WSO2 Enterprise Integrator - Message Broker Profile is developed on top of the revolutionary wso2 carbon platform and is based on the OSGi framework to achieve the better modularity for your SOA architecture. This also contains a lots of new features and many other optional components to customize the behavior of the server. Further, if you do not want any of the built in features, you can uninstall those features without any trouble. In other words, this ES can be customized to your SOA needs.
