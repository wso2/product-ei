About WSO2 EI

Product EI is an unified distribution of WSO2 Enterprise Enterprise Service Bus, Data Services Server alongside Analytics, Business Process and Message Broker and Microservice Framework for Java (MSF4J) runtimes. With the capability of shared components directory with profile based management of each runtime.

WSO2 EI 6.5.0 is developed on top of the revolutionary carbon platform (Middleware a' la carte), and is based on the OSGi framework to achieve the better modularity for your SOA architecture. This also contains a lots of new features and many other optional components to customize the behavior of the server. Further, if you do not want any of the built in features, you can uninstall those features without any trouble. In other words, this EI can be customized to your SOA needs.
